# Anki导入格式规范

## 文件格式

TSV（Tab分隔），无表头行

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 唯一递增数字 | 1, 2, 3... |
| question | 题干 + cloze占位符 | 马克思主义的基本立场是{{c1::【】}} |
| options | 选项内容（不带A./B.前缀），用\|\|分隔 | 意识第一性的\|\|物质是第一性的 |
| answer | 正确答案的**位置序号**（不是字母） | 单选: 1 / 多选: 1\|\|3\|\|4 / 判断: 1或2 |
| notes | 标签 | 第一章 / 马克思主义哲学/唯物论 |

## 题型格式

### 单选题
```
question: 题干内容{{c1::【】}}
options: 选项内容A||选项内容B||选项内容C||选项内容D
answer: 1 (对应第一个选项)
```

### 多选题
```
question: 题干内容{{c1::【】}}
options: 选项内容A||选项内容B||选项内容C||选项内容D
answer: 1||2||4 (对应第一、二、四个选项)
```

### 判断题
```
question: 题干内容{{c1::【】}}。
options: 对||错
answer: 1 (对) 或 2 (错)
```

## 导入设置

1. 打开Anki → 文件 → 导入
2. 选择txt文件
3. 勾选 **允许HTML**
4. 分隔符选择 **Tab**
5. 模板选择 **AwesomeSelect-3.x**
6. 导入

## ID规则

- 全局唯一，不可重复
- 追加牌组时，从已有最大ID + 1 开始
- 建议使用递增数字

## 标签规则

- 默认使用章节名（如"导论"、"第一章"）
- 可选AI自动生成知识点标签（如"马克思主义哲学/唯物辩证法"）
